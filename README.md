# Loan Amortization Calculator

A Java program that calculates loan payments, total interest, and the effect of extra payments on a loan.

## Features
- Calculates monthly loan payment
- Calculates total payment and total interest
- Supports extra payment scenarios
- Includes user input validation

## Technologies
- Java
- Eclipse
- Git & Github

## Files
- `Loan.java` - contains the loan calculation logic
- `LoanDemo.java` - runs the program and handles user input

## How to Run
1. Open the project in Eclipse or another Java IDE
2. Run `LoanDemo.java`
3. Enter the requested loan information in the console

## Example Inputs
- Annual interest rate: `8.25`
- Number of years: `5`
- Loan amount: `120000.95`
- Extra payment: `100`
